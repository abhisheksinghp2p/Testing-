
# #Vim
Vim has modes. This confuses beginners.

**Three Main Modes:**

| Mode         | Purpose                   | How to enter |
| :----------- | :------------------------ | :----------- |
| **Normal**   | Move around, delete, copy | Press `Esc`  |
| **Insert**   | Type text                 | Press `i`    |
| Command mode | save/exit                 | press `:`    |

### Basic Vim Steps
#### Open file:
```
vim filename.txt
```

**To type text:**
- Press **`i`** (now you can type)
- Type your text
- Press Esc when done typing
  
**To save and quit:**
- Press Esc
- Type **`:wq`**
- Press Enter
  
**To quit without saving:**
- Press Esc
- Type **`:q!`**
- Press Enter

#### Vim Cheat Sheet 
| Action                         | Keys              |
| :----------------------------- | :---------------- |
| **Start typing** (Insert mode) | `i`               |
| **Stop typing** (Normal mode)  | `Esc`             |
| **Save**                       | `:w` + `Enter`    |
| **Quit**                       | `:q` + `Enter`    |
| **Save and quit**              | `:wq` + `Enter`   |
| **Quit without saving**        | `:q!` + `Enter`   |
| **Delete line**                | `dd`              |
| **Undo**                       | `u`               |
| **Search**                     | `/word` + `Enter` |

#### Delete / Copy / Paste 
```
dd     # delete line
yy     # copy line
p      # paste
u      # undo
```

#### Search
```
/word
```

#### Save & Exit
```
:wq    # save and quit
:q!    # force quit
```


# Software management 
**installing, updating, removing**, and **verifying** software on a system

- Instead of manually downloading apps like Windows,
		Linux uses **packages + package managers**

### Package
 
 A package is a compressed file that contains:
- Program files
- Libraries
- Config files
- Metadata

## In RHEL

Package format = **.rpm** 
==**RPM**== = Red Hat Package Manager (individual packages)

**Example:**
```
httpd-2.4.57.rpm
```

#### Why Software Management is Important?

| Reason              | Explanation                      |
| ------------------- | -------------------------------- |
| Security            | Updates fix vulnerabilities      |
| Dependency handling | Auto installs required libraries |
| Stability           | Tested packages                  |
| Automation          | Fast installation                |

## Types of Tools in RHEL
#### 1. Low-level tool

**rpm**
- Works directly with .rpm files
- No dependency resolution

#### 2. High-level tool

==**dnf** (modern)==
- Handles dependencies automatically
- Works with repositories


### RPM (Low-level Tool)

- Install .rpm files
- Query package info
- Verify integrity

#### Install package
```bash
sudo rpm -ivh package.rpm
```

#### Remove package
```bash
sudo rpm -e package
```

#### Query installed packages
```bash
rpm -qa
```


#### Files in package
```bash
rpm -ql bash
```

#### ==Verify package==
```bash
rpm -V bash
```
