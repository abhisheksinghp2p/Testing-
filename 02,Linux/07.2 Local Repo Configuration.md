
# Repo
A repository = location where packages are stored

**Types:**

- Remote repo → Red Hat servers (internet)
- Local repo → ISO / local directory (offline)

## configure a LOCAL REPO from ISO
- Works without internet
- Required for RHCSA exam
- Faster (no download delay)
- Useful in secure/offline environments

## ISO
**An ISO file is:**
A disk image (like DVD)

**Contains:**

| Repo      | Contains             |
| --------- | -------------------- |
| BaseOS    | Core system packages |
| AppStream | Applications & tools |

#### When ISO is attached in VMware:

It behaves like a DVD

**Appears as:**
`/dev/sr0'

**Auto-mounted at:**
`/run/media/<user>/<ISO_NAME>`

**Example:**
`/run/media/ak/RHEL-10-1-BaseOS-x86_64`


# Two Ways ISO is Available

#### 1. Auto-mounted (GUI / VMware)

**Path:**
```
/run/media/<user>/<ISO_NAME>
```

**Example:**
```
/run/media/ak/RHEL-10-1-BaseOS-x86_64
```


#### 2. Manual Mount (Exam / Server)

**Device:**
```
lsblk
	/dev/sr0
```

**Mount to:**
```
sudo mkdir /mnt/iso
sudo mount /dev/sr0 /mnt/iso
```


## Steps to Create Local Repository
#### Step 1: Ensure ISO is Mounted

**Check:**
```
lsblk
```

**Look for:**
```
sr0  → mounted
```


##### Verify content:
```
ls /run/media/ak/RHEL-10-1-BaseOS-x86_64
```

**OR (manual mount):**
```
ls /mnt/iso
```

**Must contain:**
- `BaseOS/`
- `AppStream/`


#### Step 2: Create Repo File
```
sudo nano /etc/yum.repos.d/local.repo
```

**This file tells dnf:**
	“Where to find packages”


#### Step 3: Add Repository Configuration

**If using auto-mounted ISO:**

```
[BaseOS]
name=Local BaseOS Repo
baseurl=file:///run/media/ak/RHEL-10-1-BaseOS-x86_64/BaseOS
enabled=1
gpgcheck=0

[AppStream]
name=Local AppStream Repo
baseurl=file:///run/media/ak/RHEL-10-1-BaseOS-x86_64/AppStream
enabled=1
gpgcheck=0
```


**If using manual mount:**

```
[BaseOS]
name=Local BaseOS Repo
baseurl=file:///mnt/iso/BaseOS
enabled=1
gpgcheck=0

[AppStream]
name=Local AppStream Repo
baseurl=file:///mnt/iso/AppStream
enabled=1
gpgcheck=0
```

| Line              | Meaning                 |
| ----------------- | ----------------------- |
| `[BaseOS]`        | Repository ID           |
| `name=`           | Description             |
| `baseurl=file://` | Local filesystem path   |
| `enabled=1`       | Activate repo           |
| `gpgcheck=0`      | Disable signature check |

##### ==`file://`==

**It tells system:**
	“Use local files instead of internet”


## Refresh Repository
```
sudo dnf clean all
sudo dnf makecache
```

| Command     | Purpose            |
| ----------- | ------------------ |
| `clean all` | Remove old cache   |
| `makecache` | Load new repo data |

## Verify Repository
```
dnf repolist
```

**Output should include:**
- BaseOS
- AppStream


## Install Package (Offline)

#### Problem If using auto mount 

**Without internet:**
```
dnf install httpd
```

**Fails** (tries online repo)

**Solution:**
```
sudo dnf install httpd --disablerepo="*" --enablerepo=BaseOS,AppStream
```

| Option              | Meaning             |
| ------------------- | ------------------- |
| `--disablerepo="*"` | Disable all repos   |
| `--enablerepo=`     | Use only local repo |
